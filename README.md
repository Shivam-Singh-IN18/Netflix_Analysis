# 📺 Netflix Data Analysis

## 🚀 Overview

This project focuses on analyzing Netflix content data to uncover trends, patterns, and insights related to movies and TV shows.
The goal is to perform **exploratory data analysis (EDA)** and visualize key information to better understand content distribution and user trends.

---

## 📂 Dataset

* Netflix dataset containing information about movies and TV shows
* Includes attributes like:

  * Title
  * Genre
  * Release Year
  * Country
  * Duration
  * Type (Movie/TV Show)

---

## ⚙️ Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## 🔍 Key Analysis Performed

* Data cleaning and preprocessing
* Handling missing values
* Analysis of content type (Movies vs TV Shows)
* Year-wise content distribution
* Country-wise content production
* Genre-based analysis

---

## 📊 Visualizations

* Distribution of Movies vs TV Shows
* Content growth over the years
* Top contributing countries
* Popular genres

---

## 💡 Key Insights

* Movies dominate the Netflix platform compared to TV shows
* Significant growth in content after 2015
* Certain countries contribute more to Netflix content
* Drama and Comedy are among the most popular genres
